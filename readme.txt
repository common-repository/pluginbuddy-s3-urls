== Description ==

Allows the insertion of expiring Amazon S3 links into a post or page via a shortcode.  A template tag is also available for manual insertion (for advanced users).

<a href="https://mediaron.com/portfolio/pluginbuddy-s3-urls/">Check out the Documentation</a>.

= Features =
* Setting panel for global Amazon S3 options
* Shortcode icon for easy insertion into a post or page
* Template tag that can be used programmatically to insert S3 links anywhere

== Installation ==

1. Just unzip and upload the "pluginbuddy-s3" folder to your '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= The shortcode won't work if I don't set the expires attribute = 

This plugin is designed to insert expiring S3 links in order to protect downloads.
